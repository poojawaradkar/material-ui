# material-ui

Practice project

### Technology Used

- [React](https://reactjs.org/)
- [React Router](https://github.com/reactjs/react-router)
- [Babel](http://babeljs.io) for ES6 and ES7 magic
- [Webpack](https://webpack.js.org/) for bundling
- [Axios](https://github.com/axios/axios)

### To Run the project

- Clone the repo

```shell
https://github.com/poojawaradkar/material-ui.git
cd folder-name
```

- Installation

```shell
npm ci
```

- Running Dev Server

  1. Building Client bundle and watch for changes

  ```shell
  npm run dev
  ```
